电源指示灯：
    标识： power （D5）
    功能： 24V电源接通后M4正常工作则该指示灯闪烁，频率4s/次，占空比50%。
错误指示灯：
    标识： state（D8）
    功能： 熄灭状态表示无错误；
          长亮后闪烁1次：测试平台故障，具体地长亮1000ms，200ms后闪烁100ms
          长亮后闪烁2次：没有获取到正确的拨码开关
          长亮后闪烁3次：软件错误
模块指示灯：
    DI指示灯标识：  D17
    DO指示灯标识：  D15
    AI指示灯标识：  D11
    HIO指示灯标识： D13
    功能： 空闲状态， 指示灯闪烁，四秒一次，占空比50%。
          测试， 指示灯闪烁，一秒5次，占空比50%。
测试结果指示灯：
    标识：各个模块上方的若干LED，
    功能：显示测试结果
         长亮：表示测试通过
         熄灭：标识等待测试
         快速闪烁：表示该通道有错误。
24V开关：
    标识：
    功能：控制电源
启动测试按钮：
    标识： 
    功能： 启动测试
拨码开关:
    标识：
    功能：DI、DO、AI、HIO分别对应1、2、3、4，测试时应该将该模块对应的拨码开关拨到1

*******************************************************************************
*******************************************************************************
测试流程：
    1. 接通24V电源
    2. 安装端子板，并正确接通电缆
    3. 正确拨动拨码开关，DI、DO、AI、HIO分别对应1、2、3、4，测试时应该将该模块对应的拨码开关拨到1
    4. 点击启动按钮，有3s时间按紧端子版,3s后开始测试，模块指示灯闪烁5次/s,测试期间请按紧端子版
    5. 模块指示灯闪烁频率降低到0.25次/s后，本次测试完成，
    6. 测试结果显示在模块上方一系列led，
         长亮：表示该通道测试通过
         熄灭：表示该通道等待测试
         快速闪烁：表示该通道测试不通过
    7. 重复2-6。


*******************************************************************************
*******************************************************************************
问题及解决方案：
    1. 部分通道测试失败，现象为部分测试结果指示灯快速闪烁
        解决方案： 再次测试端子版，若仍有闪烁则端子版存在问题的可能性较大
    2. 错误指示灯亮，
     长亮后闪烁1次：测试平台故障，具体地长亮1000ms，200ms后闪烁100ms，清联系测试平台设计人员
     长亮后闪烁2次：没有获取到正确的拨码开关，设定正确即可
     长亮后闪烁3次：软件错误，清联系测试平台设计人员
